Socket SDK Kotlin/Native & Kotlin.js Example
============================================

## Prerequisites

- [Socket SDK](https://github.com/socket-sdk)
- `kotlinc-js`
- `kotlinc-native`

## Setup

Install JavaScript dependencies with `npm`:

```sh
npm install
```

Build the application and run it with `ssc`:

```sh
ssc compile -r -o .
```

Or with `npm`:

```sh
npm start
```
